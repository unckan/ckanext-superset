# CKAN Apache Superset extension

CKAN extension to connect an Apache Superset instance to CKAN.  

## Requirements

Compatibility with core CKAN versions:

| CKAN version    | Compatible?   |
| --------------- | ------------- |
| 2.10            | Yes           |
| 2.11            | not tested    |
